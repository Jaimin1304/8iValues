# 8iValues

**8iValues** is a customized version of the original [8values](https://github.com/8values/8values.github.io) political quiz, designed specifically to test and compare the political compass and ideological leanings of different Large Language Models (LLMs). 

Rather than having human users take the quiz, this project showcases pre-recorded results from various leading AI models, providing a detailed breakdown of their answers and explanations across 70 political values questions.

**8iValues** 是基于原版 [8values](https://github.com/8values/8values.github.io) 政治倾向测试的修改版，旨在专门测试和比较不同大语言模型（LLM）的政治倾向与意识形态。

与让用户自行答题不同，本项直接展示了各个主流 AI 模型在 70 道政治价值观问题上的具体回答与解释，并通过数据图表直观对比了它们的政治倾向结果。

## Features | 功能特色

- 🤖 **AI Model Benchmarking | AI 模型倾向测试**: View comprehensive political compass results for popular LLMs (Claude, Gemini, GPT, Grok, Kimi). | 查看主流大语言模型（Claude、Gemini、GPT、Grok、Kimi）的详细政治倾向测试结果。
- 📊 **Detailed Answer Logs | 详细回答列表**: Expandable UI to view the exact answers and explanations generated by each AI model for all 70 questions. | 可展开的 UI 设计，查看各个 AI 模型对全部 70 道测试题的具体回答和相应解释。
- 🌐 **Bilingual Support | 中英双语支持**: Fully localized UI with a persistent English/Chinese toggle for comparing results in your preferred language. | 完全本地化的用户界面，支持中英双语无缝切换，且状态自动保存。
- 📈 **Visual Comparisons | 直观图表对比**: Axis markers on the results page show not only the selected model's position but also the positions of other AI models for direct comparison. | 结果页的各个维度轴线上不仅展示当前模型的得分，还会标记出其他 AI 模型的得分以便直观对比。

## How to Run | 运行方法

You can run the site locally using any basic HTTP server. For example, with `npx`:

你可以使用任何基础的 HTTP 服务器在本地运行本网站。例如，使用 `npx`：

```bash
npx http-server ./ -p 8080 -c-1
```

Then open `http://localhost:8080/index.html` in your browser.

然后即可在浏览器中打开 `http://localhost:8080/index.html` 进行访问。
